# India Crime Analysis Dashboard

##  Project Overview
This project presents an end-to-end crime data analysis 
of India using Power BI. It covers crime patterns across 
major cities, age groups, gender, weapons used, and 
yearly trends from 2020 to 2024.



## Dataset
- **Source:** Kaggle — Indian Crimes Dataset
- **Link:** https://www.kaggle.com/datasets/sudhanvahg/indian-crimes-dataset
- **Type:** Synthetic dataset (used for academic purposes)
- **Records:** 40k +crimes
- **Period:** 2020–2024

---

##  Tools Used
- Power BI Desktop

##  Key Analysis Performed
- Data Cleaning & Preprocessing  
- Exploratory Data Analysis (EDA)  
- Trend Analysis   
- Crime category breakdown

- ##  Key Insights
- Delhi has the highest crime count (5.5k+ cases)
- Senior citizens are most affected age group (13K+ cases)
- All crime types are uniformly distributed (2K+ each)
- 2024 data is incomplete — sharp drop due to data cutoff
- Knife is the most used weapon (14.53%)
-  Domestic Violence is among top 5 crimes
- Females are victims of most of the crimes
